校验器(validator)

xpzheng

# 是否为外部链接

isExternalLink(url)

参数

  1. url(String): 链接

返回值

(Boolean): 判断结果

例子

isExternalLink('http://www.google.com')
// => true
isExternalLink('/users')
// => false

# 是否非空字符串

isNotBlank(str)

参数

  1. str(String): 字符串

返回值

(Boolean): 校验结果

例子

isNotBlank('')
// => false
isNotBlank('  ')
// => false
isNotBlank('123')
// => true
isNotBlank(null)
// => false
isNotBlank(undefined)
// => false
